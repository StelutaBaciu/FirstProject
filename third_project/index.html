Prima pagina HTML

HTML are numeroase etichete (tags) si elemnte populare care sunt folosite pentru a structura si prezenta continutul pe paginile web.

< reprezintă < lt= mai puțin decât > reprezintă > gt= mai mare decât Acest lucru se datorează faptului că,caracterele > și < sunt caractere „rezervate” în HTML. HTML este un limbaj de marcare < și > sunt folosite pentru a indica începutul și sfârșitul diferitelor elemente.

  • Unul dintre cele mai folosite taguri este tag-ul de,deschidere
  • Unul dintre cele mai folosite taguri este tag-ul de,inchidre
  • Tag-ul

    este folosit pentru paragafe

Un alt element popular este headingul. Headingurile pot fi de mai multe tipuri. Ele se diferentiaza prin numere.

Headingurile sunt de mai multe tipuri:

  1. <H1>
  2. <H2>
  3. <H4>
  4. <H5>
  5. <H6>

Cel mai mare heading si cel mai important este h1, si cel mai mic este h6.

Spanul este un subelement, folosit pentru a selecta si manipula un cuvant o propozitie sau o fraza.
